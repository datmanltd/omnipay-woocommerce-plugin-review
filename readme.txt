This is the official Omnipay plugin for Woocommerce that connects your woocommerce store to process payments through Omnipay payment gateway services. Omnipay offer online order processing services and data analysis to businesses throughout Europe, Omnipay are experts in online card processing and data management.						

== Description ==
Accepts Visa, MasterCard directly on your store with the Omnipay payment gateway for WooCommerce
The Omnipay payment plugin extends WooCommerce allowing you to take payments directly on your store via Omnipay's API.
Omnipay help business owners across UK achieve unbeatable rates and excellent service. Additionally, Omnipay also analyse all your statically data thoroughly, ensuring you are receiving the best service for your business and will supply recommendations on which customers you should target, and what promotional offers to send.

== Installation ==
Minimum Requirements
WooCommerce 2.1 or higher
There are two ways in which you can install the Omnipay Payment Gateway into your WordPress/WooCommerce website, Automatic and Manual.

Option 1 – Automatic Installation
This is the easiest way to install the Omnipay Payment Gateway plugin into your website. Simply log into your WordPress dashboard and navigate to the Plugins menu, then click Add New.

In the search box, type Omnipay and click Search Plugins. Once you have found the Omnipay plugin you can install it by simply clicking Install Now.

Option 2 – Manual Installation
This option requires access to your servers directory.

Unzip and upload the plugin folder to your /wp-content/plugins/ directory
Activate the plugin through the Plugins menu in WordPress
Go to WooCommerce -> Settings and click on the Checkout tab.
Find Omnipay in the Payment Gateways section
Click the settings button to configure and enable the gateway.
Choose production mode set as 'prod' will enable the omnipay to take payments using live cards
Production mode will use the below form urls
HostedForm https://gateway.omni-pay.com
BackendUrl https://kql31gmtxb.execute-api.eu-west-1.amazonaws.com/prod/api/v1/mypay
Click ‘Save Changes’.
